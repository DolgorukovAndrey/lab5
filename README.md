# Основные классы и методы
Методы и классы расположены в разных пакетах:

<h2> 1) Пакет fraction </h2>

В данном пакете расположен класс Fraction и интерфейс Methods. 

<h3> Methods </h3>
Интерфейс содержит два метода:

- float getFractionFloat()
- void setNumDen(int numerator,int denomenator)


<h3> Fraction </h3>

Описан класс Fraction (Дробь) с полями numerator,denominator, cachedFraction (вещественное значение дроби). Который наследует интерфейс Methods и встроенный Cloneable. Реализован конструктор с параметрами и без.


Методы в классе:
- setNumerator - устанавливает заданное значение числителя
- setDenominator - устанавливает заданное значение знаменателя
- getNumerator - возвращает значние числителя
- getDenominator - возвращает значние знаменателя
- clone - возвращает копию дроби
- setNumDen - устанавливает заданное значение числителя и знаменателя (из интерфейса Methods)
- getFractionFloat - возвращает вешественное представление дроби (из интерфейса Methods)
- toString - возвращает строковое представление дроби
- equals - сравнивает две дроби
- hashCode - возвращает hashCode дроби

<h2> 2) Пакет methods </h2>

В пакете представлен класс ClassMethods в котором представлены различные методы:

- deleteList - принимает список объектов и удаляет из него все объекты с заданным значением
- task4 - читает все данные из файла "file4.txt" и выводит имена людей, с наибольшей суммой баллов
- task5 - читает весь текст из файла "file5.txt" и выводит все русские согласные буквы, которые встречаются только в одном слове
- task6 - принимает очередь объектов и проверяет равенство объектов в заданном диапазоне
- task72 -  в данном методе выполняется следующее: читаются все люди из файла "file7(2).txt", все имена приводится к нижнему регистру, но с первой буквой в
верхнем регистре, убираются из перечня все люди без номеров, а имена оставшихся группируются
по их номеру и выводятся на экран

<h2> 3) Пакет point </h2>

В пакете представлены классы Point, Line, Polyline, MethodsPoint. 

<h3> Point </h3>

В данном класе представлена реализация точки. Присутствуют два целочисленных поля x и y. Реализован конструктор с параметрами.

Методы в классе:
- getX - возвращает значение координаты x
- getY - возвращает значение координаты y
- setX - устанавливает заданное значение координаты x
- setY - устанавливает заданное значение координаты y
- toString - возвращает строковое представление точки
- equals - сравнивает две точки
- hashCode - возвращает hashCode точки

<h3> Line </h3>

В данном класе представлена реализация линии. Присутствуют два поля типа Point startPoint и endPoint. Реализован конструктор с параметрами.

Методы в классе:
- getEndPoint -  возвращает точку начала
- getStartPoint - возвращает точку конца
- setEndPoint - устанавливает заданную точку начала
- setStartPoint - устанавливает заданную точку конца
- toString - возвращает строковое представление линии

<h3> Polyline </h3>

В данном класе представлена реализация ломанной линии. Присутствуют поле типа Point[] polyline в котором хранится массив точек, входящих в ломанную линию. Реализован конструктор с параметрами.

Методы в классе:
- toString - возвращает строковое представление ломанной линии

<h3> MethodsPoint </h3>

В данном классе представлен метод task7, который принимает набор объектов типа Point, берёт все Point в разных координатах, убирает с
одинаковыми X и Y, сортирует по X, отрицательные Y делает положительными и собирает это все в ломаную линию (объект типа Polyline)

<h2> 4) Пакет cat </h2>

В пакете представлены классы Cat, MeowCount, MethodsCat и интерфейс Meow.

<h3> Meow </h3>
В интерфейсе представлен метод: void meow()

<h3> Cat </h3>

В данном классе представлено описание кошки с неизменяемым после создание полем name. Класс реализует интерфейс Meow. Реализован конструктор с параметром.

Методы в классе:
- getName - возвращает имя
- meow - выводит на экран строку "имя: мяу!"
- toString - возвращает строковое представление "кот: имя"

<h3> MeowCount </h3>

В данном классе два поля: cat типа Cat и целочисленное поле countMeow для подсчёта количества мяуканий кошки. Реализован конструктор с параметром.

Методы в классе:
- getCaountMeow - возвращает количество мяуканий данной кошки
- getCatName - возвращает имя кошки из поля cat
- meow - вызывает для поля cat метод meow из класса Cat и увеличивает значение countMeow на 1

<h3> MethodsCat </h3>

В данном классе представлен метод meowing, который принимает объекты, которые умеют мяукать (в их классе реализован метод meow) и вызывает метод meow для каждого переданного объекта.

<h2> 5) Пакет main </h2>

В классе main представлена демонстрация работы методов

# Установка программы и вспомогательные файлы

<h3> file4.txt </h3>
Файл для метода task4. Представление информации в файле: <Фамилия>– строка, состоящая не более чем из 20 символов;<Имя>– строка, состоящая не
более чем из 15 символов;<Баллы>– строка, содержащая три целых числа, разделенных
пробелом, соответствующих баллам, полученным участником за каждое задание первого тура.
При этом <Фамилия> и <Имя>, <Имя> и <Баллы> разделены одним пробелом. Примеры входных
строк:

Петрова Ольга 25 18 16

Калиниченко Иван 14 19 15

<h3> file5.txt </h3>
Файл для метода task5. Файл содержит текст на русском языке.

<h3> file7(2).txt </h3>
Файл для метода task72. Представление информации в файле: файл со строками, содержащими имя человека и его номер в следующей форме:

Вася:5

Петя:3

Аня:5

<h3> Установка </h3>
Скачать все файлы из репозитория и добавить их в один java проект.
